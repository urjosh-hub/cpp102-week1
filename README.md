## Reflection
-Git and Github are useful for programming because they help developers to access and to change code and collaborate with others and to manage projects more efficiently. Git is a version control system that records every change made to a project, allowing developers rewrite the previous version of your project. Github is a cloud-based platform that host Git repositories, making it easy for multiple developers to work the same codebase review changes, and share project with others.
